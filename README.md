## for developer

### clone

```
$git clone https://github.com/phi-jp/phina.js.git
$cd phina.js
```

### setup

```
$npm install -g gulp
$npm install
```

### build

```
$gulp
```

### develop

watch and launch

```
$gulp dev
```
