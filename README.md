## for developer

### clone

```
$git clone https://github.com/phi-jp/phina.js.git
$cd phina.js
```

### setup

```
$npm install
$npm install -g gulp
```

### build

```
$gulp
```

### watch and build

```
$gulp watch
```

### launch

```
$gulp webserver
```
